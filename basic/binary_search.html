引入

二分是一个非常基础但实用性很广的算法。二分基于分治的思想，每次处理会将问题的规模减小一半。

最常见的，就是查找有序数组中的给定元素。我们可以通过比较中点和给定元素大小关系，来决定下一次比较的区间。


// 假设 a 数组递增
int find(std::vector<int> a, int v) {
    int lo = 0, hi = int(a.size()) - 1;
    while (lo <= hi) {
        int mid = (lo + hi) / 2;
        if (a[mid] == v) {
            return mid;
        } else if (a[mid] > v) {
            hi = mid - 1;    
        } else {
            lo = mid + 1;
        }
    }
    return -1;
}

我们发现，每循环一次，区间长度减半，因此这个过程的时间复杂度为 $O(n\logn)$
